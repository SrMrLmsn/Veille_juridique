# Veille.Numérique — Droit du Numérique

Site de veille juridique hébergé sur GitHub Pages, alimenté automatiquement toutes les 30 minutes via GitHub Actions.

## Sources
- CNIL, ANSSI, EDPB, Commission européenne, DPC Ireland

## Déploiement
1. Settings → Pages → Branch `main` / root
2. Settings → Actions → General → Read and write permissions
3. Actions → Fetch RSS Feeds → Run workflow (première fois)
